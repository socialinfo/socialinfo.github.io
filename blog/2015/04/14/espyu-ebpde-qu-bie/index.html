Esp与ebp的区别

Apr 14th, 2015 11:20 am | 评论

首先开门见山，引用别人的一句话

ESP就是一直指向栈顶的指针,而EBP只是存取某时刻的栈顶指针,以方便对栈的操作,如获取函数参数、局部变量等。

上一段C语言以及其的汇编代码

    int main()
    {
        test(1,2);    
        return 0;

    }
    int test(int a,int b)
    {
        int c = a + b;
        return c;
    }

    main:
        pushl   %ebp
        movl    %esp, %ebp
        andl    $-16, %esp
        subl    $16, %esp
        movl    $2, 4(%esp)  
        movl    $1, (%esp)
        call    test
        movl    $0, %eax  #return 0
        leave
        ret
    test:
        pushl   %ebp
        movl    %esp, %ebp
        subl    $16, %esp
        movl    12(%ebp), %eax
        movl    8(%ebp), %edx
        addl    %edx, %eax
        movl    %eax, -4(%ebp)
        movl    -4(%ebp), %eax
        leave
        ret

上述的汇编代码中每个函数调用时都先保存了ebp，并且将当前的esp赋值给了ebp，在函数调用完毕都恢复了ebp使用的leave指令（等价为movl %ebp, %esp　popl %ebp）

cdecl 与 stdcall的区别就是 cdecl需要自己完成函数的清除工作，而stdcall会用ret n自动完成清除工作
